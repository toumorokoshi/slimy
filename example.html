What is slimy?

Slime is an html 5 slideshow template. The slideshow is opened in-browser, and utilizes html5, javascript, and css.

Why use slimy?

  • Designed for simplicity: minimal features, and utilize existing html5/css technologies as much as possible
  • Minimal external dependencies: base is just JQuery, a style.css file, and a script.js file.

Controls

  • left arrow/h : previous slide
  • right arrow/l : next slide
  • m: menu key
  • +/-: increase or decrease the size of the slide relative to the window
  • p: preview slides

Advanced Usage

The following slides discuss some of the more rich functionality in slimy.

Custom Variables

Custom variables can be set in the html file, at the top in the head section with:

<script>
  var slide_scale = 0.5;
</script>

The following variables can be customized:

  • slide_scale: the initial size of the slide.

Slide Templates

An expiremental feature is slide templates. This allows for preset styles. For example, this slide is using the twocolomn layout.

More examples will be added into the slide_examples.html file.

To use this template, add the class "slimy_twocolumn" to your slide, and add divs with the class "slimy_left" and "slimy_right" explicitely:

	   <article class="slimy_twocolumn">
	     <div class="slimy_left">
	       LEFT STUFF
	     </div>
	     <div class="slimy_right">
	       RIGHT STUFF
	     </div>
	   </article>
