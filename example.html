Using Slimy

A tutorial on how to build slideshows with slimy

Yusuke Tsutsumi
4/4/2012

What is slimy?

Slime is an html 5 slideshow template. The slideshow is opened in-browser, and utilizes html5, javascript, and css.

Why use slimy?

  • Designed for simplicity: minimal features, and utilize existing html5/css technologies as much as possible
  • Minimal external dependencies: base is just JQuerya a style.css file, and a script.js file.

Controls

  • left arrow/l : next slide
  • right arrow/h : previous slide
  • c: menu key
  • +/-: increase or decrease the size of the slide relative to the window

Themes?

Themes can be included and contributed to themes. To use one, just include the css file!

<link rel="stylsheet" type="text/css" href="themes/YOUR_THEME.css"/>

In this example, the default style, greyhound, is being used.
