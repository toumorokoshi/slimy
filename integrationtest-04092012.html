Activating Code Coverage

Code Coverage can be activated with profiles contained in Zillow-Parent

mvn clean install -Pcobertura-instrument,get-cobertura-build,
integration-instrument -DtestIntegration
  • Cobertura-instrument: instrument cobertura files to ${instrumented.classes.dir}
  • Get-cobertura-build: download and extract build.xml file containing instructions for cobertura
  • Instrument-integration: modify integration test plugin configuration to work with instrumentation

Generating a Coverage Report

Two components must exist to have a usable coverage report: a coverage.xml (or html) file containing the actual instrumentation, then a folder containing sources (if you want to see the specific lines of code instrumented).

This performs both:

mvn -Pgenerate-cobertura-report validate
